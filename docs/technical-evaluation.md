# Evaluacion tecnica

El evaluador tecnico cubre requisitos `TECHNICAL`, `WORKFORCE`, `OPERATIONAL` y
`ORGANIZATIONAL`.

## Reglas iniciales

- Certificacion vigente: compara nombre/codigo esperado y fecha de expiracion.
- Personal o rol requerido: busca personas soportadas cuyo nombre, rol o cargo contenga el sujeto
  exigido.
- Cobertura territorial: exige coincidencia conservadora con la cobertura registrada.
- Capacidad tecnica u operativa: exige dato soportado o verificado en el snapshot.

## Criterio conservador

Certificaciones vencidas producen `DOES_NOT_COMPLY`. Capacidades declaradas sin soporte, tecnologias
no equivalentes y roles no comparables producen `UNKNOWN`.
