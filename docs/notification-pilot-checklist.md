# Checklist de piloto de notificaciones

- [ ] Consentimiento y destinatarios permitidos revisados.
- [ ] Dry-run validado antes de red.
- [ ] Kill switch probado.
- [ ] Texto, HTML, disclaimer y enlaces revisados en Mailpit.
- [ ] Firma del receptor local verificada.
- [ ] Idempotencia, 429, 400 y fallos SMTP probados.
- [ ] Límites por hora/día configurados.
- [ ] Retención ejecutada primero en dry-run.
- [ ] Sin secretos, datos reales ni evidencia sensible en logs.
- [ ] Infraestructura detenida al cierre.
