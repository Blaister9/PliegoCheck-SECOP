# Checklist de red

- [ ] Entrada HTTPS solo desde redes autorizadas.
- [ ] Sin acceso directo a DB/API/web/worker/scheduler.
- [ ] SSH administrativo conforme a política externa.
- [ ] Egress SECOP/DNS/registry mínimo; SMTP/webhook solo allowlisted.
- [ ] Firewall, VPN, DNS y proxy institucional documentados.
