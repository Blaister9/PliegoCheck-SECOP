# Checklist de seguridad

- [ ] VPN/red/allowlist y egress verificados humanamente.
- [ ] Auth activa; Secure/HttpOnly/SameSite, lockout y revocación.
- [ ] Roles mínimos, sin registro público ni bypass.
- [ ] HSTS, CSP, nosniff, frame, referrer y permissions headers.
- [ ] Sin secretos/logs sensibles; scan y revisión de imágenes aprobados.
- [ ] SSO/MFA pendientes registrados como riesgo aceptado o blocker.
