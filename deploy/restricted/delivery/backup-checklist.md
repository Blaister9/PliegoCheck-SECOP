# Checklist de backup

- [ ] Destino externo autorizado, capacidad y permisos verificados.
- [ ] Dump DB, storage, manifest, versión/schema/timestamp presentes.
- [ ] SHA-256 verificados; sin secretos, env, claves o logs.
- [ ] Lock/atomicidad/cleanup y programación probados.
- [ ] Retención dry-run revisada antes de confirmación real.
