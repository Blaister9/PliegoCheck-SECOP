# Onboarding de usuario piloto autorizado

- [ ] Identidad, rol, consentimiento y alcance aprobados.
- [ ] Cuenta nominativa; secreto entregado por canal externo.
- [ ] Cambio de contraseña y mínimo privilegio verificados.
- [ ] Avisos de no concepto jurídico/no producción explicados.
- [ ] Feedback e incidente se registran sin datos ni secretos.
- [ ] Revocación y offboarding acordados.
