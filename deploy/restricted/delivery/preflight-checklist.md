# Checklist de preflight

- [ ] Docker/Compose/OpenSSL, CPU, memoria, disco y reloj revisados.
- [ ] Puertos, DNS, rutas y permisos revisados.
- [ ] HTTPS/origins/hosts/proxies/cookies explícitos.
- [ ] Secrets files sin placeholders y con permisos mínimos.
- [ ] Certificado, hostname, vigencia y key match válidos.
- [ ] Compose, migraciones, schemas y backup tooling disponibles.
