# Documentos sinteticos de empresa

El perfil de empresa del piloto se siembra directamente como un snapshot
publicado sintetico (ver `pilot/seed/company.json` y el modulo autoritativo
`apps/worker/src/pliegocheck_worker/pilot/dataset.py`).

En una demo manual, las evidencias documentales de empresa (RUP, estados
financieros, certificaciones) se cargan por la UI de empresa con archivos
sinteticos. Este directorio queda como marcador para esos soportes ficticios;
no se versionan documentos binarios reales.
