# Nota de consentimiento y alcance

Esta sesion usa un entorno controlado con datos sinteticos. No es produccion y no debe usarse para
evaluar procesos reales, empresas reales, documentos institucionales reales ni personas naturales
reales.

La participacion implica aceptar que:

- Se observara la interaccion con fines de mejora del producto.
- Se registraran hallazgos funcionales, no datos personales reales.
- No se capturaran passwords, cookies, tokens, `.env` ni documentos reales.
- Las decisiones del sistema son preliminares y requieren revision humana.
- SSO, MFA, S3 real, OCR, PDF, firma digital e integracion SECOP estan fuera de alcance.

Si aparece dato real o secreto, la sesion debe pausarse y el hallazgo se clasifica como `BLOCKER`.
